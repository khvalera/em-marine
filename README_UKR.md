# EM-Marine Reader

**Версія:** 0.0.3.1

EM-Marine Reader — це невелика програма для роботи зі зчитувачем карток EM-Marine, який підключається через Ethernet (TCP/IP).

Програма працює в системному треї, підключається до зчитувача, отримує код картки та вводить його в активне вікно як звичайне натискання клавіш. Це зручно, коли код картки потрібно автоматично передати в іншу програму, форму, термінал, облікову систему або базу даних без ручного введення.

## Що робить програма

- Підключається до зчитувача EM-Marine через TCP/IP.
- Отримує код картки зі зчитувача.
- Вводить отриманий код у поточне активне вікно.
- За потреби може автоматично натискати **Enter** після коду картки.
- Показує стан підключення через іконку в треї.
- Веде прості журнали для діагностики.
- Працює з трею, тому основне вікно не потрібно тримати відкритим.

## Перший запуск

Під час першого запуску програма автоматично створює файл налаштувань.

У Linux файл налаштувань зберігається тут:

```text
~/.config/em-marine/options.ini
```

Типові налаштування:

```ini
[Options]
IP=192.168.1.191
Port=9761
PressEnter=No
```

Відкрийте **Налаштування** через меню в треї та вкажіть IP-адресу і порт вашого зчитувача.

## Налаштування

### IP-адреса

Мережева адреса зчитувача EM-Marine.

Приклад:

```text
192.168.1.191
```

### Порт

TCP-порт, який використовує зчитувач.

Типове значення:

```text
9761
```

### Натискати Enter

Якщо увімкнено, програма після введення коду картки додатково натискає **Enter**.

Це зручно, якщо програма, у яку вводиться код, одразу очікує підтвердження введення.

## Іконка в треї

Програма переважно працює через системний трей.

Іконка в треї показує поточний стан підключення:

- зчитувач підключено;
- зчитувач не підключено.

Натисніть правою кнопкою миші на іконку в треї, щоб відкрити меню:

- **Налаштування** — відкрити параметри підключення;
- **Про програму** — показати інформацію про програму;
- **Вихід** — закрити програму.

## Зображення та іконки

Спочатку програма шукає іконки в системному каталозі:

```text
/usr/share/pixmaps/em-marine/
```

Якщо цього каталогу немає, програма використовує каталог `images`, який знаходиться поруч із файлом запуску програми:

```text
./images/
```

Це дозволяє запускати програму без встановлення іконок у системні каталоги.

## Журнали

У Linux журнали зберігаються тут:

```text
~/.local/share/em-marine/
```

Журнали можуть допомогти знайти причину проблем із підключенням, відсутніми іконками або помилками читання картки.

## Типовий порядок роботи

1. Запустіть програму.
2. Відкрийте меню в треї та виберіть **Налаштування**.
3. Вкажіть IP-адресу та порт зчитувача.
4. Збережіть налаштування.
5. Відкрийте потрібну програму або поле введення.
6. Піднесіть картку до зчитувача.
7. Код картки автоматично з’явиться в активному вікні.

## Важливі примітки

- Програма вводить код картки як клавіатурне введення, тому потрібне вікно має бути активним.
- Якщо зчитувач недоступний, програма продовжує пробувати підключитися повторно.
- Якщо увімкнено **Натискати Enter**, цільова програма може одразу прийняти або відправити введений код.
